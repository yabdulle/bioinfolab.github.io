# MAMBAL - Machine Learning And Multiomics Bioinformatics for ALS

A modern, responsive website for the MAMBAL Lab at King's College London.

## Website Structure

### Pages

1. **index.html** - Main home page with lab overview
2. **team.html** - Complete team directory with all 24 lab members
3. **research.html** - Research areas, projects, and collaborations
4. **publications.html** - Lab publications and featured research
5. **contact.html** - Contact information and inquiry form

### Technical Files

- **css/styles.css** - External stylesheet with all CSS (dark/light themes, responsive design)
- **js/theme.js** - JavaScript for theme toggle functionality
